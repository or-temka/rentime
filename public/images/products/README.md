# Как формировать правильные изображения для продуктов

В директории продукта необходимы следующие изображения:

- `original.jpg` - оригинальное изображение товара, может быть несколько таких, к примеру: `original.jpg`, `original_1.jpg`, `original_2.png`. Формат любой.
- `full.webp` - размер приблизительно 2:1 широкоформат, можно и 16:9. Точное совпадение по имени. Используется для фона товара. Обязательно на черном фоне.
- `thumbnail.webp` - размер обязательно 3450х1980 (Можно использовать и 1920x1080). Точное совпадение по имени. Используется для карточки товара. Обязательно на черном фоне.

Можно использовать и другие фото для хранения прошлой информации в любом формате, кроме `[id предмета]_` в начале.
Фото для проекта, которые будут использоваться должны начинаться на `[id предмета]_` в начале, к примеру `playstation5_transparent_background_1.webp`.
